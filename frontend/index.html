⚙️ How to add this to Manager.io

  1. Open Manager.io and log into your business
  2. Go to Settings → Custom Buttons → New Custom Button
  3. Set Label: Goods/Services Configurator
  4. Set Endpoint: (the URL for this app — to be provided after testing)
  5. Set Placements: /inventory-items  /non-inventory-items  /inventory-item-form  /non-inventory-item-form
  6. Click Create — the button now appears on those pages

ℹ️ About

The Goods/Services Configurator implements the full URA EFRIS UNSPSC commodity taxonomy (58 segments, 547 families, 7,904 classes, 148,000+ commodities) directly inside Manager.io.


Configure items once → sync to Manager.io as Inventory (Goods) or Non-Inventory (Services) items, with full UNSPSC commodity codes stored in custom fields for URA EFRIS compliance.
